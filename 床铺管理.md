新建床铺接口为：POST /api/beds

下面是接口的详细定义

```
POST /api/beds
提交参数例子：
{
    "roomId": 8,
    "bedNo": "1-下",
    "bedType": 2,
    "status": 0
}

提交参数：TypeScript数据格式
/**
 * Request
 *
 * BedUpsertRequest
 */
export interface Request {
    bedNo?: string;
    /**
     * 1上 2下
     */
    bedType?: number;
    /**
     * 编辑时带
     */
    id?: number;
    /**
     * 新建时必须
     */
    roomId?: number;
    /**
     * 0空闲 1占用 2锁定 3待打扫
     */
    status?: number;
    [property: string]: any;
}


返回数据例子：
{
  "code": 0,
  "message": "",
  "data": 0,
  "timestamp": 0
}


返回数据：TypeScript数据格式
/**
 * Request
 *
 * ResultLong
 */
export interface Request {
    /**
     * 业务码：约定 0=成功，非0=失败
     */
    code?: number;
    /**
     * 返回数据
     */
    data?: number;
    /**
     * 提示信息
     */
    message?: string;
    /**
     * 时间戳（毫秒）
     */
    timestamp?: number;
    [property: string]: any;
}
```



##  技术栈

- 前端框架：Vue 3.3+
- UI组件库：Element-Plus
- 状态管理：Pinia
- 路由：Vue Router 4
- HTTP请求：Axios
- 语言:  TypeScript 5.2+
